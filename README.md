# Antifraud Desk: Сигнал

Браузерная мини-игра о работе специалиста антифрода.  
Запоминай последовательность тревожных сигналов на рабочем столе аналитика — и повторяй их до последнего рубля.

## Как запустить

```
Откройте index.html в любом современном браузере (Chrome / Firefox / Edge / Safari).
Сервер не нужен — игра работает локально без backend.
```

## Управление

| Действие | Как |
|---|---|
| Запустить | Открыть `index.html` |
| Ввести позывной | Текстовое поле на стартовом экране |
| Выбрать режим | Кнопки «Одиночный» / «Дуэль» |
| Играть | Кликать по приборам в фазе ввода |
| Сброс ввода | Кнопка ↺ в нижней панели |
| Подстава (дуэль) | Кнопка 💣 — добавляет 2 ложных сигнала противнику (1 раз за матч) |

## Правила

1. **Фаза наблюдения** — приборы подсвечиваются дважды. Запоминай только истинные сигналы, игнорируй слабые ложные вспышки.
2. **Фаза ввода** — кликай по приборам в том же порядке. Есть таймер (7 + номер раунда сек).
3. **Стресс-шкала** — каждый ошибочный клик заполняет её. При 100% — провал раунда.
4. **Баланс** — начальный 1 000 000 ₽. Штраф за ошибку: 10 000 × номер раунда. При 0 — конец.

## Технологии

- Чистый HTML5 / CSS3 / ES6 JavaScript
- Web Audio API (звук без файлов)
- localStorage (таблица рекордов)
- Без зависимостей, без сервера
